# FYP202526-Lin-AI-CV-Builder
Final Year Project (CT413): An AI-Powered CV builder and enhancement tool for STEM graduates in Ireland and the UK, focusing on content quality, ATS optimisation, and job-specific tailoring.

## Project Overview
This project is the Final Year Project for the module CT413. The aim of the project is to design and develop an AI-powered CV builder and enhancement tool aimed primarily at STEM graduates.

The system focuses on improving CV content quality rather than layout alone. It provides intelligent guidance to help users write clearer, achievement-focused, and ATS-friendly CVs, with optional tailoring to specific job descriptions.

## Problem Context
Many job applicants struggle to effectively communicate their skills and achievements in a concise and structured CV. This challenge is amplified by the widespread use of Applicant Tracking Systems (ATS), which filter CVs before they reach human recruiters. Existing CV tools often prioritise templates and formatting rather than content improvement.

## Project Objectives
- Design an interactive CV editor with structured sections
- Provide AI-assisted suggestions to improve CV content clarity and impact
- Support ATS-aware optimisation and keyword alignment
- Allow users to tailor CVs to specific job descriptions
- Export the final CV in a professional format

## Technologies (Initial Plan)
- Frontend: React with TypeScript
- Backend: Python (FastAPI)
- AI/NLP: Large Language Model API
- Data Storage: Local storage / lightweight database (initially)
